# Системпромпти AI-асистента хакатону

Один універсальний протокол + картка на кожну задачу. Асистенту команди
подається **два файли**: `SYSTEM_PROMPT.md` і картка її треку.

| файл | що це |
|---|---|
| `SYSTEM_PROMPT.md` | універсальний протокол: валідація → дослівна трансляція → «стоп і запитай», заборони (інтернет, `*/admin/`), анти-джейлбрейк |
| `tier_a.md` | картка tier_a (6DOF attitude): **можна редагувати все, крім admin/** |
| `tier_b.md` | картка tier_b (CTBR, PPO vs SAC): діри — reward + гіперпараметри |
| `tier_c.md` | картка tier_c (Blender FPV): єдиний файл — `solution.py` |
| `tier_d.md` | картка tier_d (табличний Q-learning): діри у starter.ipynb + team_solution.py |

Ця тека — організаторська: у роздачу команді йде лише `SYSTEM_PROMPT.md` +
картка її задачі (вбудовані в системний промпт асистента), не всі картки.
