# Realistic Drone Simulator in Blender

Реалістичний симулятор польоту FPV-квадрокоптера, інтегрований у середовище Blender. Містить як інтерактивний ручний політ з клавіатури, так і автономний режим відтворення траєкторії, розрахованої бекенд-алгоритмами планування шляху.

---

## 🛠️ Нові можливості

1. **Режими польоту (Ручний / Автономний)**:
   * **Ручний**: Двигун фізики реального часу (`RealisticQuad`) з керуванням клавішами.
   * **Автономний**: Попередня симуляція руху дрона за допомогою алгоритмів планування шляху (**A*** та **APF**) та 2D-лідара. Blender виконує плавне відтворення (Replay) розрахованих кадрів.
   * Перемикач режимів повністю інтегровано в бокову N-панель (вкладка «Дрон»). Перемикання негайно скидає старий політ і респавнить дрон на старті.

2. **Ігрова 3POV (Chase) камера**:
   * Камера слідує за координатами дрона та його курсом (`yaw`), але **не завалюється** по крену (`roll`) та тангажу (`pitch`). Це імітує камеру від третьої особи з комп'ютерних ігор та виключає тряску картинки.

3. **Інваріантність до розкладки клавіатури**:
   * Керування повністю незалежне від мови введення. Гарячі клавіші працюють однаково стабільно як на англійській, так і на українській / російській розкладках (`W/Ц`, `S/І/Ы`, `A/Ф`, `D/В`, `Q/Й`, `E/У`, `R/К` для скиду, `C/С` для камери).

4. **Генерація випадкових карт для всіх**:
   * Звичайні учасники (без прав адміна) тепер можуть генерувати випадкові мапи в обох режимах. У меню панелі відображається поточний `seed` карти.

5. **Освітлення та вимкнення розмітки Blender**:
   * Перемикання в'юпорту у режим **RENDERED** (Eevee) додає м'яке 3D-освітлення сонця з реалістичними динамічними тінями.
   * Будь-яка службова розмітка Blender (сітка підлоги, значки камер, ламп, Empty-об'єкти, рамки виділення) автоматично **вимикається** при старті польоту, перетворюючи в'юпорт на чисте вікно гри.
   * Вибір об'єктів мишкою заблоковано (`hide_select`), тому випадковий клік не збиває фокус камери.

---

## ⌨️ Керування (Ручний політ)

* `W` / `S` — тангаж (pitch)
* `A` / `D` — крен (roll)
* `Q` / `E` — курс (yaw)
* `Space` / `Shift` — газ (throttle)
* `C` — перемикання камери (FPV / Chase від 3-ї особи)
* `R` — скинути дрон на старт мапи (рестарт)
* `Esc` — вихід з поточного польоту

---

## 🚀 Як запустити

Відкрийте термінал у папці проекту та запустіть Blender зі скриптом:

```bash
/Applications/Blender.app/Contents/MacOS/Blender -P blender_manual.py
```

*Примітка: для роботи скрипта Blender використовує вбудований Python. Усі бібліотеки (`numpy` тощо) підключаються автоматично з оточення.*
